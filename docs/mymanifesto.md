# Tsumory マニフェスト

「積もる」と「story」を掛け合わせた名前。一粒ずつ積もったつぶやきが、ひとつの物語になる。

## コンセプト

忙しくて日記を書く時間もエネルギーもない人のためのサービス。長い文章を書かせない。SNSに一言つぶやく感覚で投稿するだけで、1日の終わりにAIがその日の日記を書き上げてくれる。「書く」のではなく「積もらせる」ことが日記になる。

## ユーザー体験

- 思ったこと・あったことを、SNS感覚でその都度一言ずつ投稿する(推敲しない、まとめない)。
- 1日の終わりに、その日の投稿をAIが1本の日記として自動生成する。
- 出来上がった日記が「しっくりこない」と感じたら、何度でも作り直せる。納得のいく1本に出会うまで気軽にやり直せることが、押し付けがましくない体験につながる。

## AIのトーン

書き手ではなく、そばで話を聞いていた人のように書く。断定的に決めつけず、その日のつぶやきに寄り添いながら、本人の言葉や温度感を尊重して物語に仕上げる。上から評価したり説教したりしない。押しつけがましくなく、何度でも作り直せる前提だからこそ、気軽に肩の力を抜いたトーンでよい。
